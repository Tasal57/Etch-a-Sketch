# Etch-a-Sketch

## Description
Etch-a-Sketch is a web-based drawing application built using HTML, CSS, and JavaScript. 
Users can draw on a grid by hovering over cells and change the size of the grid for different drawing experiences.

## Features
- Dynamic grid drawing
- Adjustable grid size
- Clear button to reset the grid
- Interactive and responsive design

## Technologies Used
- HTML5
- CSS3 (Grid, Flexbox)
- JavaScript (DOM manipulation)
- Git & GitHub for version control
